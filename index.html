Features

Built for depth, not speed

Every design decision optimizes for understanding quality over processing time.

Multi-Agent Architecture

Multiple AI agents collaborate — reading code, extracting relationships, validating facts, and generating visuals. No single-prompt shortcuts.

Schema-Validated

Every node and edge in the knowledge graph conforms to a strict JSON schema. Structured data, not free-form text.

Source-Linked

Every fact traces back to the exact file and line. Click any node to see the code that produced it.

Interactive Visual Output

The output is a self-contained HTML presentation with diagrams, timelines, and explorable knowledge graphs. No login required.

Deep Analysis

Agents spend hours, not seconds. They read every file, trace call graphs, understand architecture patterns, and map domain concepts.

Zero Dependencies

Output is a single static HTML file. No frameworks, no build tools, no servers. Open it in any browser and explore.
